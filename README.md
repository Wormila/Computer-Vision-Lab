# Computer-Vision-Lab
The experiments focus on core computer vision operations such as image acquisition, transformations, histogram processing, edge and feature detection, enhancement, filtering, texture segmentation, image segmentation, and object detection. All tasks are implemented using OpenCV to gain practical understanding of vision algorithms.
# Introduction
This laboratory introduces core Computer Vision concepts through hands-on experiments using Python and OpenCV. It covers image processing, feature detection, segmentation, and object detection techniques essential for real-world vision applications.
# Aim
To gain practical knowledge of image and video processing methods by implementing Computer Vision algorithms using OpenCV.

# Objectives
To acquire hands-on experience with OpenCV

To understand image analysis techniques

To apply vision algorithms to real images and videos

# Software and Tools Requirements
A computer system with Python 3.x installed along with OpenCV, NumPy, Matplotlib, and optional deep learning libraries. Webcam is required for video experiments.

# Theory
The experiments apply mathematical and algorithmic concepts on images using OpenCV. These methods enhance image quality, detect features, analyze textures, and identify objects, forming the basis of intelligent vision systems.

# Algorithms
Standard Computer Vision algorithms such as Canny, Harris, SIFT, Gabor filtering, segmentation, and deep learning-based detection are implemented using OpenCV functions.

# Applications
Computer Vision applications include object detection, image enhancement, texture analysis, motion analysis, medical diagnostics, and smart security systems.

# Steps of Execution of Program
Install dependencies

Read input image or video

Execute vision algorithm

Visualize results

Terminate program

# Results
Expected outputs such as enhanced images, detected edges, segmented regions, and identified objects were obtained, confirming correct implementation.

# Conclusion
Through these experiments, essential Computer Vision algorithms were implemented successfully, strengthening practical skills and conceptual knowledge in visual computing.
